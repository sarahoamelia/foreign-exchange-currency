## Getting Started
- [To install packages]
- `npm install`
- [To run the project]
- `npm start`
- [To build from docker]
- `docker build -t sarahamelia/fec .`
- [To running from docker]
- `docker run -it   -v ${PWD}:/usr/src/app   -v /usr/src/app/node_modules   -p 3000:3000   --rm   sarahamelia/fec`
